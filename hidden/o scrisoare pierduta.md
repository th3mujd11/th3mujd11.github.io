# O scrisoare pierduta
## de I.L. Caragiale

---

### Intro

O scrisoare pierduta, comedia lui IL Caragiale, este incadrabila in perioada clasicilor literaturii romane si este o creatie cu certe aspecte realiste.

Piesa lui Caragiale abordeazao tematica sociala si zugraveste un amplu tablou al degradarii vietii politice, sociale si private. Autorul alege sa aduca in prim-plan un episod din campania electoriala pentru functia de deputat.

In stilul comediei, Caragiale dezvolta aceasta tema prin prezentarea unor situatii hazlii in care sunt implicate persoanje ce sunt definite prin defecte caracteriale.

Intriga piesei este o comedia de moravuri in care, prezentand contrastul dintre aparenta si esenta autorl satirizeaza defectele omenesti.

### Specifice dramaticului/comediei

Din punct de vedere compozitional piesa este alcatuita din 4 acte si lista de personaje implicate ofera o serie de detalii referitaore la personaje, statul acestora si relatia dintre ele.

Actiunea piesei este plasata intr-un orasel de provincie, in zilele noastre, reperele de spatiu si de timp fiind nedeterminate, sugerand idea unei generalizari a situatiei si mai cu seama idea repetabilitatii moravurilor.

Intriga piesei este antrenata de pierderea nei scrisori compromitatoare, care ajunge la adversarul politic si devine o arma de santaj.

Conflictul dramatic este politic si se desfasoara intre doua laturi politice:cea a prefectului judetului, Tipatescu si cea a avocatului nae Catavencu, cel care isi doreste mandatul de deputat. Asadar, acest tip de conflict este un conflict de interese politice, personajele luptandu-se pentru binele personal si nu pentru binele societatii. Politica este deplasata dinspre scopul social catre scopuri meschine, autorul criticand astfel coruptia societatii.

Aceest conflict este alimentat de-a lungul piesei printr-o serie de actiuni urmand schema bulgarului de zapada. Conflictului principal i se adauga cel dintre Farfridi si Branzovenescu aflat in conflict cu Tipatescu si Trahanache. Un alt conflict este cel dintre gruparea locala si cei de la centru.

Ca in orice comedie, conflictul se sfarseste fericit, toata lumea se impaca la o festivitate condusa de Catavencu, acesta demonstrand ca miza confruntarii a fost superficiala si ca cei lipsiti de caracter pot coexista usor.
